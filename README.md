# README

A barebone starter kit using Webpack and Backbone

## Prerequisites
- NPM

## Usage

To install:
`npm install`
`npm install webpack webpack-dev-server -g`

To run:
`npm start` or `webpack-dev-server`

Then visit:
http://localhost:8080/webpack-dev-server/

To build:
`npm build` or `webpack`
